# Changelog

All notable changes to Dark City RPG will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Environment variable configuration system
- Performance monitoring and caching
- Enhanced accessibility features
- Comprehensive error handling
- Security improvements for PHP APIs

### Changed
- Consolidated configuration files
- Improved character loading with caching
- Enhanced error messages and user feedback

### Security
- Removed hardcoded API keys and webhook URLs
- Added API key validation to all endpoints
- Implemented proper CORS configuration
- Added input sanitization and validation

### Fixed
- Character card styling issues
- Error handling in character loading
- Accessibility improvements for keyboard navigation

## [1.0.0] - 2024-11-27

### Added
- Initial release of Dark City RPG game hub
- Character creation and management system
- Real-time character submission and moderation
- Community calendar integration
- Responsive design for mobile devices
- GitHub Pages deployment configuration
- WebSocket integration for real-time updates
- Character gallery and profile pages
- Server-side API with Node.js/Express
- MongoDB integration for data storage
- Discord webhook notifications
- Comprehensive character templates
- Rule documentation system
- Playbook system for character classes

### Security
- JWT authentication system
- Rate limiting for API endpoints
- Input validation with Joi
- Helmet middleware for security headers
- CORS configuration
- Environment-based configuration

### Performance
- Lazy loading for images
- API response caching
- Optimized database queries
- Minified CSS and JavaScript
- Efficient DOM manipulation

### Accessibility
- WCAG 2.1 AA compliance
- Screen reader support
- Keyboard navigation
- Focus management
- ARIA labels and landmarks
- High contrast mode support
- Reduced motion support

### Documentation
- Comprehensive README
- API documentation
- Deployment guides
- Character creation templates
- Game rules documentation
